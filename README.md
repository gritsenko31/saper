## ğŸ® How to Play

### Objective
Uncover all safe squares without clicking on any mines. Use the numbered clues to deduce where mines are hidden [web:37][web:39].

### Controls

**Desktop:**
- **Left Click** - Reveal a square
- **Right Click** - Flag/unflag a suspected mine
- **Both Click** (on number) - Chord: reveal all adjacent unflagged squares [web:32]
- **Smiley Button** - Start new game

**Mobile:**
- **Tap** - Reveal a square
- **Long Press** - Flag/unflag a square [web:37]
- **Tap Smiley** - New game

### Understanding Numbers
Each number shows how many mines touch that square (including diagonally) [web:32][web:37]:
- **1** - One mine in the 8 surrounding squares
- **2** - Two mines adjacent
- **8** - All surrounding squares contain mines (rare!)
- **Blank** - No adjacent mines (auto-reveals neighbors)
